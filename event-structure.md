Event Structure
====

Multi Day Event
===

__Friday Evening__
* 19:00-20:00 - Introduction of people and projects
* 20:00 - Break out and start working
* 21:00 - Late Dinner / Snacks
* 22:00 - Status Update
* 06:00 - Status Update

__Saturday__
* 06:00 - Status Update
* 08:00 - Breakfast
* 12:00 - Lunch
* 14:00 - Status Update
* 18:00 - Dinner
* 22:00 - Status Update

__Sunday__
* 06:00 - Status Update
* 08:00 - Breakfast
* 12:00 - Lunch
* 14:00 - Progress Reports & Demo

Single Day Event (ie a Saturday)
===
* 10:00-11:00 - Introduction of people and projects
* 11:00 - Break out and start working
* 12:00 - Lunch
* 14:00 - Status Update
* 18:00 - Dinner
* 22:00 - Progress Reports & Demo
